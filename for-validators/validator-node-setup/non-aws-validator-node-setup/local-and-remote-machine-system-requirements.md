## **系统要求**

* Ubuntu 16.04 系统
* Minimum 1 核心CPU
* Minimum 1GB 内存
* Anything &gt; 4GB 会很好，但将来可能需要升级

**控制机器依赖性**

* Linux Based Bash Terminal
* Python 2 \(v2.6-v2.7\)/Python3 \(v3.5+\)
* Ansible v2.3+
* Git
